## Getting the Data
You can download the data from [MovieLens here](https://grouplens.org/datasets/movielens/) (25M dataset)


## Preparing the Data
```sh
python ./movie-finder/make_dataset.py
```

## Running the App
```sh
streamlit run ./movie-finder/app.py
```
